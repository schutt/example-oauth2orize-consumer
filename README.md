## Usage

## Install

```bash
npm install -g jade

git clone https://github.com/woshiwanting/example-oauth2orize-consumer.git

cd example-oauth2orize-consumer/

npm install

jade public/*.jade

```

## Start

```bash
node server.js
```

```
open http://localhost:3001
```
